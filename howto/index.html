fishTones.js: a JavaScript toolkit to bring LC-MS/MS data to your browser

fishTones.js is a JavaScript library to visualize Liquid Chromatography Tandem Mass Spectrometry data in modern web browsers.

Tired to tediously access to data, one experiment at a time, on proprietary software? fishTones.js offers a large variety of features and graphical widgets to build powerful mass spectrometry browser applications. More than a single minded end user software, fishTones.js is a toolkit to shape a large variety of client applications. Originally designed to fish histone peptides (that's etymology), the library has been used in many complex situations.

Visit the getting started page to kickoff a project and learn more through documentation and practical examples. Four chapters cover theoretical computations (dry), experimental data (wet), aligning both (match) and more.

This how-to is not aimed at documenting comprehensively fishTones.js library. The goals is to demonstrate the basic principles and explain design choices, to enlighten the key API calls and illustrate them by short code snippets. A full documentation can be found in the source code and, more importantly, throughout the test suites. Technical notes are detailed in the github README.md file.
