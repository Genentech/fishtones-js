Using fishTones.js

NB: throughout theses pages, we will demonstrate the usage of the library via its bundled distribution (one file packaged with all its dependencies). It is also possible to use it in a lighter, more dynamic and more elaborated way with require.js. More information on the development perspectives can be found on github.

NB': the library has been developed and tested on modern web browser, target mainly at Chrome, Firefox and Safari. Some features may not be available on older navigators.

What can you find on theses pages?

This JavaScript mass spectrometry library has been organized in three sub structure, mimicked by the current documentation:
  • dry: the theoretical computations of peptide structures, masses etc.
  • wet: experimental data such as mass spectra, XIC etc.
  • match: alignment of wet/dry or wet/wet information.
Each part contain models (object representation), collections (lists or dictionaries of objects), services (single point handling data access, or computation factories) and views (rendering visualization, and managing interactions).

Documentation structure

Aimed to be an overview of the library, the current documentation tries to cover three majors aspects:
  • documentation presents the design choices, the key components;
  • (partial) API offer a summary of the main methods, whereas source code or unit tests are more comprehensive;
  • code (+ viz) show small code snippets and their actual output. The actual mechanism is described below.

How do code snippet actually work?

Each code snippet, visible when clicking on the button, is loaded from a .html and a .js files. The JavaScript is actually executed for the demo, meaning that the sample code is a fully working example. The mechanism under this features is stored in the sampleCode jQuery plugin with howto-utils.js script.

Finally, code samples take advantage of two mainstream libraries: jQuery.js ($) and underscore.js (_). The mention to $('$target') in the Javascript sources is replaced at runtime by the container of the example html template.

How to actually use fishTones.js?

To use fishTones.js in your own web page, the simplest way is to include the JavaScript bundle and css into your header. Both files can be downloaded in the github dist/ directory. You can then later use the exposed fishtones object within a <script> element or in an included .js file.

NB: if you need to include your own jQuery or Bootstrap libraries, it must be done after the fishtones-js-bundle-min.js import, to avoid overloading effects.

<html>
<head>
    <link rel="stylesheet" type="text/css" href="css/fishtones-js.css"/>
    <script src="lib/fishtones-js-bundle-min.js"></script>
</head>
<body>
    <script>
        var peptide = new fishtones.dry.RichSequence().fromString('PKPTIDER');
    </script>
</body>
</html>
